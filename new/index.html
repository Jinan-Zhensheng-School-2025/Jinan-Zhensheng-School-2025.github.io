如何获取 Token？

  1. 打开 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 点击 Generate new token (classic)
  3. 勾选 repo 权限（完整仓库访问）
  4. 生成后复制 Token 并粘贴到上方输入框

Token 仅保存在你的浏览器本地，不会上传到任何服务器。
